# Manual Tests

This directory contains manual test scripts for various components:

- `test_deepseek.py` - DeepSeek API integration tests
- `test_deepseek_embedding.py` - DeepSeek embedding functionality tests
- `test_fallback.py` - Fallback mechanism tests
- `test_model_selection.py` - Model selection logic tests
- `test_official_api.py` - Official API tests

These are standalone test scripts that can be run independently for debugging and validation.
